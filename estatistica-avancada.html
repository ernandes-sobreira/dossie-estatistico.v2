Guia de uso
🔍 Post-hoc — quando usar
Execute depois de uma ANOVA significativa. A ANOVA diz "há diferença entre os grupos" — o post-hoc diz quais pares diferem.

Tukey HSD: preferido quando os grupos têm tamanhos similares. Controla bem o erro tipo I.
Bonferroni: mais conservador. Preferido quando há poucas comparações ou quando você definiu as comparações a priori.
📐 Shapiro-Wilk — limites
Funciona bem para n entre 3 e 50. Para n grande (> 100), o teste rejeita normalidade mesmo com desvios mínimos sem impacto prático — use o Q-Q plot visualmente.

Atenção: p > 0,05 não "prova" normalidade — apenas não há evidência suficiente para rejeitá-la.
📏 Tamanho de efeito
Cohen's d (2 grupos): pequeno ≥ 0,2 · médio ≥ 0,5 · grande ≥ 0,8
eta² (η²) (k grupos): pequeno ≥ 0,01 · médio ≥ 0,06 · grande ≥ 0,14

Tamanho de efeito complementa o p-valor — um efeito pode ser significativo mas biologicamente irrelevante (n grande) ou relevante mas não significativo (n pequeno).
🔥 Correlação — cuidados
|r| < 0,3 = fraca · 0,3–0,7 = moderada · > 0,7 = forte

Correlação não implica causalidade. Variáveis podem co-variar por confundimento, sazonalidade ou casualidade.
Use Pearson para dados normais e lineares. Use Spearman para dados não normais, ordinais ou com outliers.
