Guia — Bloco 3
🌿Índices de Diversidade
Shannon H' — mede diversidade considerando riqueza e equitabilidade. Quanto maior, mais diversa. Típico: 1,5–3,5 para comunidades naturais.
Pielou J' — equitabilidade: 0 = uma espécie domina tudo, 1 = todas as espécies com igual abundância.
Simpson D — probabilidade de dois indivíduos escolhidos aleatoriamente serem da mesma espécie. Use 1−D (diversidade de Simpson).
Margalef d — riqueza relativa ao tamanho amostral: d = (S−1)/ln(N). Comparável entre amostras de tamanhos diferentes.
Rarefação — padroniza o tamanho amostral ao menor N do conjunto. Essencial para comparar comunidades com N muito diferentes.
🔭PCA — interpretação
PC1+PC2 ≥ 70% — biplot confiável para interpretação.
Loadings — correlação entre variável e componente. |loading| > 0,6 = contribuição forte.
Biplot — setas = variáveis (loadings), pontos = amostras (scores). Setas longas = variável importante. Ângulo entre setas ≈ correlação entre variáveis (0° = r≈+1, 90° = r≈0, 180° = r≈−1).
Scree plot — escolha o número de PCs pelo "cotovelo" — onde a variância para de cair abruptamente.
Padronização — sempre padronize quando as variáveis estão em unidades diferentes (pH, temperatura, condutividade). Sem padronização, variáveis de maior magnitude dominam.
